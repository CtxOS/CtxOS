# Security Policy

## Supported Versions

Only the latest version of CtxOS is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

We take the security of CtxOS seriously. If you discover a vulnerability, please report it privately to the maintainers.

**Please do not open a public GitHub issue for security vulnerabilities.**

### Process
1. Email your findings to: `security@ctxos.io` (Note: This is a placeholder email).
2. Include details of the vulnerability and steps to reproduce.
3. We will acknowledge your report within 48 hours.
4. We aim to provide a fix within 30 days for confirmed critical vulnerabilities.

### Scope
- Core toolkit scripts
- Software Center backend (DBus/Polkit)
- Build pipeline artifacts (GPG signing, etc.)

*Thank you for helping keep CtxOS secure!*
